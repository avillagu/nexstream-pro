TAREA 3: IMPLEMENTAR PROXY DE THUMBNAILS

TAREA 4: IMPLEMENTAR EXTRACCION DE INFO
DESCRIPCIÓN: Endpoint que usa yt-dlp para extraer metadatos del video
AGENTE RESPONSABLE: Backend Developer
INPUTS: Arquitectura, formato de respuesta esperado
OUTPUTS: Endpoint GET /api/info
PASOS:
1. Recibir URL via query param
2. Ejecutar yt-dlp --dump-json --no-playlist
3. Parsear JSON de salida
4. Retornar: title, thumbnail, duration, platform
CRITERIOS DE ACEPTACIÓN: Retorna JSON válido con metadatos del video
DEPENDENCIAS: Tarea 2
ESTADO: Pendiente

---

TAREA 5: IMPLEMENTAR DESCARGA Y CONVERSION
DESCRIPCIÓN: Endpoint principal que descarga y convierte video a MP4 compatible
AGENTE RESPONSABLE: Backend Developer
INPUTS: Arquitectura, config de ffmpeg
OUTPUTS: Endpoint GET /api/download
PASOS:
1. Recibir URL via query param
2. Spawning yt-dlp con selector de formato 720p
3. Pipe stdout de yt-dlp a stdin de ffmpeg
4. ffmpeg escribe archivo temporal MP4 con faststart
5. Enviar archivo temporal al cliente via stream
6. Limpiar archivo temporal
7. Manejar cierre de conexión del cliente
CRITERIOS DE ACEPTACIÓN: Descarga MP4 reproducible compatible con WhatsApp
DEPENDENCIAS: Tarea 2
ESTADO: Pendiente
